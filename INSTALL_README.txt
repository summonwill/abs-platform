ABS Platform - AI Bootstrap System
===================================

Welcome to ABS Platform!

This installer will guide you through setting up ABS Platform on your computer.

WHAT IS ABS PLATFORM?
---------------------
ABS Platform is an AI-powered project management and development assistant that helps you:
• Manage projects with AI assistance
• Maintain session notes and context automatically
• Work with multiple AI providers (OpenAI, Anthropic, Google)
• Keep governance documentation up-to-date

REQUIREMENTS
------------
• Windows 10 or Windows 11 (64-bit)
• Internet connection (for AI features)
• API key from OpenAI, Anthropic, or Google (you can set this up during installation or later)

SETUP WIZARD
------------
The installer will ask you to:
1. Choose an installation location
2. Select your preferred AI provider
3. Enter your API key (optional - can be done later in Settings)
4. Choose a default AI model

Your API key is stored LOCALLY on your computer and is never transmitted to anyone except the AI provider you choose.

GETTING AN API KEY
------------------
If you don't have an API key yet, you can get one from:
• OpenAI: https://platform.openai.com/api-keys
• Anthropic: https://console.anthropic.com/
• Google: https://makersuite.google.com/app/apikey

Click Next to continue with installation.
